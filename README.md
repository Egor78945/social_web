Проект Социальная сеть, содержит такие команды, как:
- Отправлять сообщения другим пользователям
- Постить / удалять новости
- Лайкать / дизлайкать
- Комментировать / удалять комментарий
- Отправлять дружеский запрос / отменять дружеский запрос
- Принимать дружеский запрос / отклонять дружеский запрос
- Блокировать / разблокировать пользователя (Для пользоватей с ролью Admin)
